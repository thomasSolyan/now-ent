Colors

The following colors have been selected to compliment the essence and personality of the brand. Only the shown colors should be used. The brand’s main color, along with shades of orange and grey, will service as the foundation of the palette. The secondary colors are to be used only as a supplement to the primary colors. For a more subtle contrast, lighter and darker shades of these base colors are used throughout the site design. They should be used to add depth and variety. The main blue color and all of it’s variations are used in both header text and backgrounds.
