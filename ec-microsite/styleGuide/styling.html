Text Styling

Standard Text

The global default font-size is 14px, with a line-height of 1.428. This is applied to the <body> and all paragraphs. In addition, <p> (paragraphs) receive a bottom margin of half their computed line-height (10px by default).

Nullam quis risus eget urna mollis ornare vel eu leo. Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Nullam id dolor id nibh ultricies vehicula.

Cum sociis natoque penatibus et magnis dis parturient montes, nascetur ridiculus mus. Donec ullamcorper nulla non metus auctor fringilla. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit. Donec ullamcorper nulla non metus auctor fringilla.

Maecenas sed diam eget risus varius blandit sit amet non magna. Donec id elit non mi porta gravida at eget metus. Duis mollis, est non commodo luctus, nisi erat porttitor ligula, eget lacinia odio sem nec elit.

Make a paragraph stand out by adding the .lead class.

Small text

For de-emphasizing inline or blocks of text, use the <small> tag to set text at 85% the size of the parent. You may alternatively use an inline element with .small in place of any <small>.

This line of text is meant to be treated as fine print.

<small>This line of text is meant to be treated as fine print.</small>

Big text

For large emphasized inline or blocks of text, use the .big class to set text at 125% the size of the parent.

This line of text is meant to be treated as Big text.

<p class="big">This line of text is meant to be treated as Big text.</p>

Bold

Use the <strong> tag for emphasizing a snippet of text with a heavier font-weight.

The following snippet of text is rendered as bold text.

<strong>rendered as bold text</strong>

Italics

Use the <em> tag for emphasizing a snippet of text.

The following snippet of text is rendered as italicized text.

Available glyphs

200 glyphs icons in font format are available. For performance reasons, all icons require a base class and individual icon class. To use, place the following code just about anywhere. Be sure to leave a space between the icon and text for proper padding.

<span class="glyphicon glyphicon-[icontype]"></span>
