Branding Guidelines

This site is a detailed guide to assist in the production, implementation and continuing development of the Blue Cross Blue Shield family of websites. This sites primary goal is to set rules for code consistency and best practices. This guide outlines the overall design, coding guidelines, styles, file structure, naming conventions and resources used in the construction of this site. To ensure design continuity and maintain the life span of this site, it is important that everybody involved in the development, maintenance and updating of this site read, understand and use these guidelines. By adhering to best practices, we ensure optimized page loading, performance and maintainable code.

Colors

The following colors have been selected to compliment the essence and personality of the brand. Only the shown colors should be used. The brand’s main color, along with shades of orange and grey, will service as the foundation of the palette. The secondary colors are to be used only as a supplement to the primary colors. For a more subtle contrast, lighter and darker shades of these base colors are used throughout the site design. They should be used to add depth and variety. The main blue color and all of it’s variations are used in both header text and backgrounds.

Fonts

Arial is to be used as the main typeface for standard text. Arial is a widely available sans-serif typeface and computer font packaged with Microsoft Windows, other Microsoft software applications and Apple Mac OS systems. When used on navigation elements, selected elements, and emphasized text Arial Bold is to be used for improved legibility.
